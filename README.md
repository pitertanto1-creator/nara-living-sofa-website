\# Nara Living - Website Katalog Sofa



Nara Living adalah website katalog produk sofa untuk bisnis furniture/UMKM. Website ini dibuat untuk menampilkan produk sofa secara profesional, responsif, dan mudah dihubungi melalui WhatsApp.



\## Fitur Website



\- Tampilan responsive untuk laptop dan HP

\- Katalog 10 produk sofa

\- Fitur pencarian produk

\- Filter kategori produk

\- Modal detail produk

\- Galeri gambar produk

\- Tombol WhatsApp untuk pemesanan

\- Desain modern dan elegan untuk bisnis furniture



\## Teknologi yang Digunakan



\- HTML

\- CSS

\- JavaScript
